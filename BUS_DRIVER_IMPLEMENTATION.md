# BUS_DRIVER_IMPLEMENTATION.md — Dina Mobile

> **Scenario (from Dina README):** User is waiting at a bus stop. User's Dina contacts the Bus Driver's Dina. "What time will you reach my location (XYZ)?" BusDriver's Dina checks its internal database + Google Maps traffic and answers "45 minutes".
>
> This requires **D2D service query + response**, **durable workflow tasks**, **schema-validated delegation to OpenClaw**, **AppView-backed public service discovery**, and **a review/approval path** for sensitive capabilities.
